Description du projet

Ce projet de ePortfolio a été conçu comme un CV vivant, permettant de mettre en valeur mon parcours académique et profesionnelle, mes compétences, ma vision en technopédagogie et mes réalisations clés. Le ePortfolio est structuré de manière à offrir une navigation fluide, une hiérarchie visuelle claire et une expérience interactive agréable, autant pour les recruteurs que pour les collaborateurs potentiels.

Outils utilisés

Pour concevoir ce projet, j’ai utilisé HTML5, CSS3 (Flexbox, Grid), JavaScript léger pour les interactions, ainsi que la bibliothèque AOS (Animate On Scroll) pour les animations. La conception a été faite en suivant les principes UX/UI, avec un accent particulier sur la responsivité mobile. les images ont été générées par ChatGPT suite Une description littérale détaillée sous forme de prompt ou instruction. Le code est hébergé sur GitHub Pages.

Défis rencontrés

Les principaux défis ont été de concevoir une structure responsive, d'assurer la cohérence visuelle entre les sections, d'intégrer des animations sans alourdir la performance, et de proposer une interface qui respecte les principes d'accessibilité et d'expérience utilisateur. Un autre défi majeur a été le choix des outils : j'ai d'abord exploré plusieurs CMS comme WordPress, Google Sites et même Wix. Toutefois, ces solutions ne permettaient pas d’atteindre le niveau de personnalisation et de précision visuelle que je recherchais. C’est pourquoi j’ai finalement opté pour une programmation complète à partir de zéro. Par ailleurs, la présentation synthétique de mes différents projets académiques constituait aussi un défi : il s’agissait de structurer l’information de manière claire, hiérarchisée et engageante, sans surcharger la navigation. Enfin, la gestion des médias, l'équilibre entre contenu et design, ainsi que l’optimisation du chargement ont également été des enjeux techniques importants.
